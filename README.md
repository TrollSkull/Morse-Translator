# Morse Translator
- Translate to Morse and vice versa
- Made in Python 3

## Instalation
	$ pkg install -y git

	$ pkg install -y python

	$ pkg install -y python3

	$ git clone https://github.com/TrollSkull/Morse-Translator

	$ cd Morse-Translator

	$ python3 translator.py
