# Morse Translator
- Translate to Morse and vice versa
- Made in Python 3

Version 1.2

## Instalation
	$ pkg install -y git

	$ pkg install -y python

	$ git clone https://github.com/TrollSkull/MorseTranslator

	$ cd MorseTranslator

	$ python translator.py
